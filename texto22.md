# HTML - Elemento BODY

## Funções e Contexto

O elemento `<body>` é fundamental em HTML, pois ele representa o **conteúdo visível** de um
documento HTML. Tudo o que o usuário vê (textos, imagens, links, vídeos, etc.) é colocado dentro
deste elemento.

* **Conteúdo Visível:** Ele engloba todo o conteúdo que é exibido na janela do navegador.

* **Hierarquia:** Ele é filho direto do elemento `<html>` e é o irmão do elemento `<head>`.

**Exemplo Básico:**

```HTML

<!DOCTYPE html>
<html>
<head>
  <title>Título da Página</title>
</head>
<body>
  <h1>Olá, Mundo!</h1>
  <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

```
