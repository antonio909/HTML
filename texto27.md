# HTML - Tag TITLE

A tag `<title>` em HTML é usada para **definir o título do documento HTML**. Ele é um elemento
de **metadados** e é sempre colocado dentro da seção `<head>` do seu documento.

## Onde aparece?

O conteúdo da tag `<title>` é exibido em locais importantes para o usuário e motores de
busca:

* **Barra de Título ou Aba do Navegador:** É o texto que aparece na aba ou barra de título da
  janela do navegador.

* **Resultados de Busca:** É o texto principal (o link clicável) exibido nos resultados de uma
  pesquisa (como Google, Bing, etc.).

* **Favoritos/Bookmarks:** É o texto padrão usado quando um usuário adiciona sua página aos
  favoritos.

## Importância

A tag `<title>` é **crucial** por dois motivos principais:

1. **Acessibilidade e Usabilidade:** Ajuda os usuários a identificar rapidamente o conteúdo da
   página, especialmente aqueles que usam tecnologias assistivas (como leitores de tela) e ao
   navegar entre múltiplas abas.

2. **SEO (Search Engine Optimization):** É um dos fatores mais importantes para o
   ranqueamento nos motores de busca, pois ajuda a definir o tópico principal da página.
   Recomenda-se que o título seja descritivo, exclusivo (dentro do seu site) e tenha até cerca
   de 55-60 caracteres para não ser cortado nos resultados de busca.

-----

## Exemplo de Uso

```HTML

<!DOCTYPE html>
<html>
<head>
  <title>Minha Primeira Página Web - Aprenda HTML Aqui</title>
</head>
<body>
</body>
</html>

```
