# O que são Linguagens de Marcação?

## Introdução às Linguagens de Marcação

Linguagens de marcação são sistemas para anotar
texto de forma sintaticamente distinguível do texto.
Elas são usadas para adicionar metadados a um
documento, fornecendo informações sobre a
estrutura, a formatação e, às vezes, até mesmo a
semântica do conteúdo. Essa "marcação" fornece
instruções ao software sobre como o texto deve ser
processado ou exibido. Ao contrário das linguagens
de programação que *executam* instruções, as
linguagens de marcação *descrevem* os dados.

### O conceito central: anotação de texto

Em sua essência, uma linguagem de marcação
envolve a adição de tags ou outras anotações ao
texto. Essas tags funcionam como instruções,
informando ao software (por exemplo, um
navegador web, um processador de texto ou um
sistema de composição) o que fazer com o texto
que elas contêm. O próprio texto original é
considerado o "conteúdo". A marcação adiciona
*estrutura* e *significado* a esse conteúdo bruto.

### Principais características das linguagens de marcação

* **Legibilidade:** Embora sejam destinadas
  principalmente a máquinas, as linguagens de
  marcação são normalmente projetadas para
  serem pelo menos um pouco legíveis por
  humanos. Isso permite que os desenvolvedores
  inspecionem e editem a marcação diretamente.
* **Orientado à estrutura:** as linguagens de
  marcação são excelentes para definir a
  estrutura de um documento, como títulos,
  parágrafos, listas, tabelas e outros elementos.
